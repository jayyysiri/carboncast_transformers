# Carbon Intensity Forecasting for ISO New England with Transformers
**Authors: Ben Healy, Jay Siri, Sidhant Kumar**

This repository contains the code for our Spring 2026 1.C51 project, including:
- data loading and pre-processing
- transformer model implementation and training
- evaluation and comparisons to CarbonCast CNN-LSTM
- uncertainty quantification via deep ensembling
- feature importance understanding via SHAP

The original data and baseline modeling for this project were sourced from [CarbonCast (Maji *et al.*, 2022)](https://github.com/carbonfirst/CarbonCast).
